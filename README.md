# Demo Realtor

This is a demonstration project showcasing a premium, luxury real estate web application. It features a modern, brutalist architectural design aesthetic, smooth cinematic page transitions, and interactive floating property filters designed to emulate high-end real estate portals.

## Features
- **Cinematic Hero Display:** Fluid hero sections that elegantly bleed into bento-grid layouts.
- **Glassmorphic Floating Filters:** Intuitive embedded search overlays with seamless, real-time filtering updates.
- **Dark Architecture Theme:** A strictly adhered-to "Quiet Luxury" and brutalist architecture visual theme.
- **Next.js 15 App Router:** Cutting-edge React server components and seamless hydration.
- **MongoDB Integration:** Connected real estate listing data parsed directly into the application.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

This project is a design and UX capability demonstration and functions entirely as a demo platform.
