# Brand Guidelines: Demo Realtor

## 1. Aesthetic Direction
- **Primary Style:** "Architecture Brutalist" mixed with "Quiet Luxury."
- **Visual Reference:** The Woodland site (Dark theme, expansive whitespace, heavy serif typography).
- **Core Concept:** Immersion over information. The site should feel like a digital gallery.

## 2. Design Intelligence Hook (UI-UX Pro Max)
Antigravity must use the `ui-ux-pro-max` skill to generate the `MASTER.md` using the following parameters:
- **Product Type:** Luxury Real Estate / Architectural Service.
- **Keywords:** Dark mode, Cinematic, Premium, Minimalist, Bento-Grid, Serif.
- **Tech Stack:** Next.js + Tailwind.

## 3. Interaction Design
- **Hover States:** Soft image scaling (1.05x) and depth-based shadows.
- **Transitions:** Layout-ID animations for filtering (no abrupt refreshes).
- **Typography:** - **Headings:** High-contrast Serif.
  - **Meta Data:** Wide-spaced, mono-spaced or clean sans-serif.

## 4. Anti-Patterns (Avoid)
- No "AI purple" or neon gradients.
- No standard 3-column "card" rows (Use bento only).
- No harsh easing functions; use `cubic-bezier(0.33, 1, 0.68, 1)`.
