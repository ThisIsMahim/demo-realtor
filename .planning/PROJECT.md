# Demo Realtor

## Context
A luxury real estate platform built with Next.js 15, MongoDB, and Tailwind CSS. Employs "Architecture Brutalist" and "Quiet Luxury" aesthetics, inspired by 'The Woodland' site, leveraging a Dark mode Bento-Grid layout. It aims for immersion over information, feeling like a digital gallery.

### Active
- [ ] Responsive, dark-themed bento grid layout
- [ ] Fully interactive property filters (Style, Area, Price) using Layout-ID animations
- [ ] Kinetic typography and weighted scroll animations using GSAP and Framer Motion
- [ ] Integrated Lenis smooth scrolling
- [ ] Verified MongoDB schema with optimized indexing for realtor listings
- [ ] Image assets via Cloudinary for responsive, high-res presentation

### Out of Scope
- No "AI purple" or neon gradients.
- No standard 3-column "card" rows (Use bento only).
- No harsh easing functions; use `cubic-bezier(0.33, 1, 0.68, 1)`.
